## Dataset Setup

The **RML2016.10a dataset** is required to run this task. Due to repository size limits, the dataset file is not included in this commit.

1. **Download**: Please download the `RML2016.10a_dict.pkl` file from the official DeepSig website or the following source:
   - [DeepSig Datasets](https://www.deepsig.ai/datasets) (Look for RADIOML 2016.10A)
   
2. **Installation**: Place the downloaded file into the `data/` directory of this task.
